# Hospital Gungeon – Flujo de Escenas (v1)

## Flujo del juego

Start
 → Menu
 → Preload
 → Narrative
 → Hub
 → Sector (ER)
 → Hub
 → Sector (Surgery)
 → Hub
 → Sector (Labs)
 → Hub
 → Sector (Psych)
 → Hub
 → Sector (Basement)
 → Hub
 → Boss
 → Ending
 → Menu

## Notas
- El jugador avanza presionando ENTER (placeholder).
- El RunManager decide cuándo hay Sector, Boss o Ending.
- Las escenas solo reaccionan a eventos.
